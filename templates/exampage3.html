Problem Statement

Write a program to calculate the sum of all even numbers in a given list. If there are no even numbers, the output should be 0.

Input Format

The first line contains an integer n, the number of elements in the list (1 ≤ n ≤ 100).The second line contains n space-separated integers (−1000 ≤ each number ≤ 1000).

Constraints

  • The input list will always contain at least one element.
  • Each number in the list will be an integer.

Expected Output

A single integer representing the sum of all even numbers in the list.
