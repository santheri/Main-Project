Problem Statement

Vikas is trying to find words that can be formed out of given characters. A string is good if it can be formed by characters from chars (each character can only be used once). Return the sum of lengths of all good strings in words.

Input Format

First line contains integer n (no of strings). Second line contains n space-separated strings. Third line contains string chars.

Constraints

  • 1 ≤ n ≤ 104
  • 1 ≤ chars.length ≤ 106
  • Strings contain only lowercase English letters

Expected Output

Print the sum of lengths of all good strings in words.
